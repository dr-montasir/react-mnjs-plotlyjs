## React demo project using mnjs together with plotly.js

### Get started

Clone app & install the dependencies...

```bash
git clone https://github.com/dr-montasir/react-mnjs-plotlyjs.git
cd react-mnjs-plotlyjs
npm install
```

Start:

```bash
npm start
```
